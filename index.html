Restaurant Bill Calculator
A fully functional console-based billing system built in C++. Customers can browse a menu of food and drink items, place orders with quantities, apply discount schemes, and receive a formatted receipt that is also saved to a .txt file automatically.
C++ OOP Inheritance Polymorphism File Handling Console App

// 02 — oop concepts

OOP Concepts Used

01
Classes & Objects
8 classes: MenuItem, FoodItem, DrinkItem, Order, OrderItem, Customer, Bill, Restaurant
02
Inheritance
FoodItem and DrinkItem both inherit properties and methods from the base class MenuItem
03
Polymorphism
calculatePrice() behaves differently — FoodItem uses flat price, DrinkItem uses size multiplier
04
Encapsulation
All data is private, accessed only through public getter methods — data is protected
05
Abstraction
MenuItem is an abstract class with pure virtual functions, hiding implementation details
06
File Handling
Every receipt is saved automatically as a timestamped BILL-*.txt file using ofstream
